# Simple-Quiz-App

A simple and interactive quiz application built using HTML, CSS, and JavaScript.

## Features

- Randomly selects 4 questions from a question bank
- Multiple-choice questions
- Instant answer feedback
- Score tracking
- Play Again functionality
- Beginner-friendly code structure
- Displays HTML tags like `<a>`, `<img>`, and `<h1>` correctly as text

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Project Structure

```
quiz-app/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How It Works

1. The quiz starts automatically.
2. Four random questions are selected from the question bank.
3. Users select an answer.
4. Correct and incorrect answers are highlighted.
5. The final score is displayed after all questions are answered.
6. Users can restart the quiz using the "Play Again" button.
